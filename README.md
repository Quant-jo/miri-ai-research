## 📊 이슈 테마 분석 — Issue Impact Analysis

하나의 뉴스/이슈가 **포트폴리오 전체**에 미치는 영향을 실시간 분석합니다.

- 수평 영향도 게이지로 자산별 수혜/피해를 직관적으로 시각화
- 자산별 영향도 점수 (-100 ~ +100) 정량화 + 구체적 사유 제시
- **찐수혜주 발굴** — 내가 놓치고 있는 숨겨진 수혜 종목 AI 추천 (프리미엄)
